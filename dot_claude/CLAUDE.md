# コーディング

## コード全般

- 実装は必要十分に留め、「念のため」の冗長な記述を避けること。これはエラーハンドリングや fallback に限らず、CSSプロパティの重複指定、重ねがけの型ガード、二重のバリデーション等、あらゆる領域に適用される。問題の原因と必要なメカニズムを正しく理解した上で、最小の介入で目的を達成すること。期待通りに動かない場合の対処は「別の手段を併用する」ではなく「メカニズムの理解を見直す」が原則。多層防御はミッションクリティカルな場面に限る。
